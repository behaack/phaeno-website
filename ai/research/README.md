# Research notes

Use this folder for reusable evidence behind scientific, product, investor, regulatory, or competitive claims. Do not store secrets, customer data, or copyrighted source documents here.

Each note should include:

- question and intended page/audience;
- source title, publisher, URL or repository path, and access date;
- whether the source is primary, peer-reviewed, regulatory, company-authored, or secondary;
- concise findings in original words;
- exact claim the evidence supports and material limitations;
- approval status and reviewer when copy is sensitive.

Research is evidence, not automatically approved website copy. Material scientific or commercial claims should be reviewed before publication, and superseded notes should link to the newer evidence rather than being silently rewritten.
